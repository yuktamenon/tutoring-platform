# Peer-to-Peer Tutoring Platform

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: PostgreSQL
- ORM: Prisma

## Features
- Student & Tutor roles
- Tutor profile creation
- Subject assignment
- Availability management
- Booking system (request → accept → complete)
- Messaging system
- Review & rating
- Tutor recommendations

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
